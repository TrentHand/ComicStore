What is this website all about?

Comic Book Store is an ecommerce platform built from Django on the back-end with Bootstrap for the design elements on the front-end. Originally, the data for the comics were collected from the Marvel API. I have since stored it in a basic json file so that pulling it down from the GitHub Repo will include everything you need to reconstruct the website.
